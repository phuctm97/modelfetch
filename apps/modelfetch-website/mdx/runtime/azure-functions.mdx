The `@modelfetch/azure-functions` package lets you deploy MCP servers to [Azure Functions](https://azure.microsoft.com/en-us/products/functions).

## Installation

```npm title="Terminal"
npm install @modelfetch/azure-functions
```

## Usage

```typescript title="src/index.ts"
import handle from "@modelfetch/azure-functions";
import server from "./server.js"; // Import your McpServer

// Run as an Azure Functions app
handle(server);
```

## API Reference

### `handle(server)`

Starts the MCP server

- **server**: Required [`McpServer`](https://github.com/modelcontextprotocol/typescript-sdk?tab=readme-ov-file#server) instance from [`@modelcontextprotocol/sdk`](https://github.com/modelcontextprotocol/typescript-sdk)
